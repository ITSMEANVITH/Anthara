# Anthara - Event Management & Ticket Booking System

Anthara is a full-stack Event Management & Ticket Booking System built using Java, JSP, Servlets, JDBC, MySQL, Maven, and Apache Tomcat following the MVC architecture.

## Features
- User Authentication
- Event Discovery
- Ticket Booking
- Online Payments
- QR Ticket Generation
- Organizer Dashboard
- Admin Dashboard

## Tech Stack
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Maven
- Apache Tomcat
- HTML
- CSS
- Bootstrap
- JavaScript

## Architecture
MVC + DAO Pattern
